AWS服务器测评：1CPU/1GB内存/7GB硬盘/EC2服务器

发表于 2020-06-28

​ Amazon Web Services (AWS)亚马逊云计算服务，由亚马逊公司所创建的云计算平台，向个人、企业和政府提供一系列包括信息技术基础架构和应用的服务，如存储、数据库、计算、机器学习等等。AWS提供的大多数服务都使用按需付费的收费模式，按照用户使用资源的级别和时长收费。

​ AWS提供免费的EC2实例，每月提供最多750小时的运行时间，15GB的流出流量，对于小流量玩家来说是绰绰有余。下面跟随小编来看一下如何创建EC2服务器吧~

注册

​ 官网：https://aws.amazon.com/cn/

点击创建AWS账户 注册账号。

​ 值得一提的是，这里需要需要手机号验证。实测国内号码可以通过验证，只是验证码接收时间可能有延迟。

提交完会让你提交付款信息，实测虚拟卡可以通过验证。

这里小编使用了中国银行发行的万事达莫奈信用卡通过了验证。

为了防止流量超出导致扣费，小编这里建议将验证付款信息的信用卡/虚拟卡锁死。

​ 之后需要验证手机号。小编这里使用Google Voice虚拟手机号通过了验证。

​ 验证通过以后会收到账户已就绪的邮件。

接下来就可以登录控制台了。

登录界面如图。

开通机器

控制台如图所示。

这里选择启动虚拟机.

​ 注意：创建虚拟机之前记得右上角选好地区。如果开通香港地区的机器，需要先启用香港地区。

小编这里开通的是东京的机器。

注意选择免费套餐，不然会出现莫名扣费！

小编这里选择的Ubuntu18.04.

实例类型选择符合条件的免费套餐t2.micro。

点击下一步：添加存储.

点击下一步：添加标签.

继续选择配置安全组.

接下来是配置防火墙。小编一般喜欢允许所有流量，打开所有端口，在服务器上来管理防火墙。

最后一步审核和启动。

确认无误点击启动。

EC2默认启用仅密钥方式登录。小编这里推荐这种方式，可以避免SSH被他人恶意登录。

点击下载密钥对。

将下载的私钥导入到SSH登录管理页面中，小编这里使用的是Termius。该软件是收费的，使用通过Github Education Pack的Github账户可以免费使用1年。

导入完毕后选择启动实例。

找到IP之后就可以在Termius中登录了。

​ 注意：使用Ubuntu操作系统，默认登录的用户名是ubuntu，Debian系统是admin。
